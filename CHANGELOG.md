## [Unreleased]

### Added

- **Dynamic block pattern** for ECR and AgentCore following [terraform-aws-ecs](https://github.com/terraform-aws-modules/terraform-aws-ecs) best practices
- New object variable `ecr_image_scanning_configuration` for flexible image scanning configuration
  - Supports setting `scan_on_push` via object
  - Set to `null` to completely omit the configuration block
  - tfsec compliant
- New object variable `ecr_encryption_configuration` for flexible encryption configuration
  - Supports `encryption_type` (`AES256` or `KMS`) and optional `kms_key`
  - Works alongside existing `create_ecr_kms_key` and `ecr_kms_key_arn` variables
- New object variable `runtime_lifecycle_configuration` for runtime lifecycle settings
  - Supports `idle_runtime_session_timeout` and `max_lifetime`
  - Set to `null` to omit the lifecycle configuration block entirely
- Dynamic `timeouts` block for flexible timeout configuration
